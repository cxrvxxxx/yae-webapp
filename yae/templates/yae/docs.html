Getting started
Yae Miko — Discord Bot
Yae Miko is a Discord Bot built using Python & Discord.py library. It includes a multitude of features allowing you to manage your server using commands from the chat box as well as security functionality that protects your server from unwanted spams, trolls, and many other security risks making your server a safer place for its members.

Usage instructions
Using Yae Miko
Yae Miko can receive commands from any text channels it has access to. To send a command, the default prefix is "y!" but it can be changed to any prefix of your preference later using a command. The format for text-based commands are as follows
PrefixCommand Argument(s)
Example:
y!echo Hello!
Where "y!" is the prefix, "echo" is the command, and "Hello!" is the argument.

Modules
Admin Module
This module contains commands that allows you to manage the operation of the bot. These commands require elevated permissions by default.

Command Description
setprefix prefix Set the prefix for the current server to the one specified.
yaechannel Set the current channel in the current server to be used only for Yae Miko commands. Invalid commands and regular chat will automatically be deleted.
setaccess @user 0-5 Set the mentioned user's access level to the one specified. This allows access to certain commands that require elevated permissions.
sudo Gives the user the highest access level available.
restart Restars the bot. Restricted to the bot owner only.
sync optional: -c/-g Sync application commands. Use with "-c" to clear, "-g" to sync globally, or none to sync locally.
echo message Makes the bot repeat the message.


Info Module
This module contains commands that displays information about other modules such as subcommands and usage specifications.

Command Description
info Shows information about Yae Miko.
admin Shows information about the Admin Module.
spam Shows information about the Anti-spam Module.
music Shows information about the MusicPlayer Module.
extras Shows information about the Extras Module.
slash Shows information about the Slash Commands Module.


Anti-spam Module
This module prevents spam. Currently there are no commands present in this module and operates by listening to message events in the server.

MusicPlayer Mdodule
This module contains commands related to playing music in voice channels in the server. (Will be deprecated soon, replaced by slash commands.

Command Description
join Makes the bot join the user's current voice channel.
leave Makes the bot leave the voice channel.
play title/url Play the specified song/url in the voice channel.
stop Stop playback in the current channel and clears the queue.
pause Pause playback in the current channel.
resume Resume playback in the current channel.
skip Play the next song in the queue.
prev Play the previously played song.
vol Set the player's volume.
fave optional:title Add the currently playing song to your list of favorites. If a title is specified, add that instead.
unfave index Removes the song specified with the idnex argument from your list of favorites.
playliked optional:index Add your list of favorite songs to the queue. If an index is specified, only add that instead.


Slash Commands Module
This module contains a list of application commands. These commands do not use the standard prefix syntax and instead use "/" as a prefix.

Command Description
whois @user Shows information about the mentioned user.


Extras Module
This module contains commands that do not classify in any of the categories above.

Command Description
coinflip Flip a coin
